# DiologBotMemory

Telegram-бот-помощник с долговременной памятью на основе векторной базы данных Pinecone.

## Описание

Бот ведёт диалог с пользователем и запоминает всё, что тот рассказывает о себе. При каждом новом сообщении бот извлекает из памяти релевантные факты и использует их в ответе — создаётся эффект «живой» памяти, сохраняющейся между сессиями.

Перед записью нового факта система вычисляет косинусное сходство с уже сохранёнными данными: дубликаты и семантически близкие фразы не записываются повторно, что экономит место в индексе и снижает шум.

## Технологии

| Категория | Инструмент |
|---|---|
| Язык | Python 3.10+ |
| Telegram API | [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI) |
| LLM | OpenAI API (GPT-4o-mini) |
| Эмбеддинги | OpenAI `text-embedding-3-small` |
| Векторная БД | [Pinecone](https://www.pinecone.io/) |
| Конфигурация | python-dotenv |

## Реализованный функционал

- Диалог с пользователем через Telegram с ответами от LLM
- **Краткосрочная память** — скользящий буфер последних 20 сообщений, передаётся в контекст модели
- **Долговременная память** — факты пользователя сохраняются в Pinecone, при каждом сообщении достаются 5 наиболее релевантных
- **Дедупликация по косинусному сходству** — перед записью проверяется сходство с существующими фрагментами; при сходстве ≥ порога запись пропускается или обновляет существующий слот
- Изоляция памяти разных пользователей через метаданные `user_id`
- Команды управления памятью (`/clear`, `/forget`)
- Логирование всех операций записи в Pinecone (попытка / записано / пропущено / ошибка)

## Инструкция по запуску

**1. Клонировать репозиторий и создать виртуальное окружение**

```bash
python -m venv venv
venv\Scripts\activate       # Windows
# source venv/bin/activate  # Linux / macOS
```

**2. Установить зависимости**

```bash
pip install -r requirements.txt
```

**3. Заполнить `.env`**

```env
PINECONE_API_KEY=ваш_ключ
PINECONE_INDEX_NAME=название_индекса
OPENAI_API_KEY=ваш_ключ
OPENAI_BASE_URL=https://api.openai.com/v1   # или URL прокси
TELEGRAM_BOT_TOKEN=токен_бота
CHAT_MODEL=gpt-4o-mini                      # опционально
```

**4. Запустить бота**

```bash
python bot.py
```

**5. (Опционально) Проверить данные в Pinecone**

```bash
python pinecone_manager.py
```

Скрипт найдёт все записи по слову «Марс» во всех namespace'ах индекса и выведет их в терминал.

## Настройка дедупликации

В начале файла `pinecone_manager.py` доступны три глобальные переменные:

```python
COSINE_SIMILARITY_THRESHOLD: float = 0.80   # порог сходства (0.0 – 1.0)
COSINE_DEDUP_ACTION = "skip"                 # "skip" — пропустить, "update" — обновить слот
COSINE_DEDUP_ENABLED: bool = True            # включить / выключить проверку
```

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Приветствие, краткая справка по боту |
| `/clear` | Очистить краткосрочный буфер текущего диалога |
| `/forget` | Удалить всю долговременную память о пользователе из Pinecone |
